import { Aside, Tabs, TabItem, MdxRepl } from 'astro-pure/user'

组件可以让你在站点中一致地复用同一块 UI 或样式。不仅能在 `.astro` 文件中使用，也可以在 `.mdx` 文件中使用。

<Aside type='tip'>
[MDX](https://mdxjs.com/) 是一种可在 Markdown 中嵌入 JSX 的格式，其余与普通 Markdown 无异。
</Aside>

在 `.astro` 中可直接导入并使用组件，首节会给出示例。

<Aside type='tip'>
若经常在内容中引用某些组件，可使用 [astro-auto-import](https://github.com/delucis/astro-auto-import/tree/main/packages/astro-auto-import) 自动导入，无需在每篇文件中手动 import。
</Aside>

### MDX Repl

<MdxRepl>
  <MdxRepl width='100%'>
  <p>Hello</p>
  <Fragment slot='desc'>
    ```html
    <p>Hello</p>
    ```
  </Fragment>
  </MdxRepl>

<Fragment slot='desc'>
  ````jsx
  import { MdxRepl } from 'astro-pure/user'

  // Width is optional; using width parameter to set
  // all elements inside the MDX Repl component.
  // (slot desc is not included this parameter)
  <MdxRepl width='100%'>
  <p>Hello</p>
  <Fragment slot='desc'>
    ```html
    <p>Hello</p>
    ```
  </Fragment>
  </MdxRepl>
  ````
</Fragment>
</MdxRepl>

插槽 `desc` 内可组合任意其他组件。`<Fragment>` 不会渲染为实际的父级 HTML 容器。

## 资源与图标

### Icon 图标

<MdxRepl width='100%'>
<div>
import { Icon } from 'astro-pure/user'

单次使用：<Icon name='rss' class='inline' />

预览所有可用图标（点击按钮复制名称）：

import { Icons as allIcons } from 'astro-pure/libs'

<div class='flex flex-wrap gap-2'>
{
  Object.keys(allIcons).map(icon => {
    const script = `navigator.clipboard.writeText('${icon}');document.dispatchEvent(new CustomEvent('toast',{detail:{message:'已复制 "${icon}"'}}))`
    return (
      <Button as='button' type='button' class='cursor-pointer' onClick={script}>
        <Icon slot='before' name={icon} />
        <span>{icon}</span>
      </Button>
    )
  })
}
</div>
</div>
