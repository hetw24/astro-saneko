### **卸载**
备注:此教程来源于[Nodeseek-Retr0](https://www.nodeseek.com/post-208653-1) 
**当你的机器不再需要探针时,如何优雅停止nezha service呢?**

**停止agent：**
```shell
systemctl stop nezha-agent
```

**取消开机自动启动：**
```shell
systemctl disable nezha-agent
```

**删除agent二进制文件：**
```shell
rm -rf /opt/nezha
```

**删除service：**
```shell
rm /etc/systemd/system/nezha-agent.service
```
